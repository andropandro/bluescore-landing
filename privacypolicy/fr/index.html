Politique de Confidentialité

Votre vie privée nous tient à cœur

Dernière mise à jour : 26 mai 2025

Aperçu

BlueScore est développé dans le respect de votre vie privée. Nous ne collectons que les données nécessaires au fonctionnement de l'application et stockons toutes les informations localement sur votre appareil.

Informations Collectées

  • Recherches de lieux que vous effectuez
  • Données météo pour les lieux sélectionnés
  • Paramètres et préférences de l'application
  • Données de journal techniques pour le dépannage et le support

Informations Non Collectées

  • Informations personnellement identifiables
  • Informations de compte ou données utilisateur
  • Images, fichiers média ou contenu personnel
  • Données de localisation continues en arrière-plan
  • Messages personnels ou communications

Fichiers de Journal et Dépannage

L'application crée automatiquement des fichiers de journal sur votre appareil pour aider au dépannage et au support. Ces fichiers de journal contiennent :

  • Informations techniques sur les fonctions de l'application
  • Messages d'erreur et données de performance
  • Horodatages des activités
  • Informations sur l'appareil (version iOS, version de l'application)

Un nouveau fichier de journal est créé à chaque démarrage de l'application et un maximum de cinq fichiers de journal précédents sont sauvegardés automatiquement. Les fichiers de journal ne contiennent AUCUNE information personnelle comme les noms, les lieux visités ou les paramètres personnels. Les informations sensibles comme les clés API sont filtrées automatiquement.

Partage des Fichiers de Journal

Vous pouvez choisir volontairement de partager les fichiers de journal avec notre équipe de support via les paramètres de l'application. Cela ne se produit que lorsque vous choisissez activement d'envoyer un e-mail avec les fichiers de journal en pièces jointes. Aucun fichier de journal n'est envoyé automatiquement ou sans votre consentement.

Gestion des Données

Toutes les informations sont stockées localement sur votre appareil. Aucune donnée n'est envoyée à nos serveurs ou partagée avec des tiers, sauf lorsque vous choisissez volontairement de partager les fichiers de journal pour le support.

Services Tiers

Nous utilisons les services externes suivants pour récupérer des informations météo et de lieu :

  • MET Norway API (données météo)
  • GeoNames API (informations de lieu)
  • OpenStreetMap/Overpass API (données géographiques)
  • WeatherAPI.com (données météo supplémentaires)
  • Apple Maps & CoreLocation (informations de lieu)
  • Autres services de données publiques

Ces demandes sont effectuées sans partager vos informations personnelles. Seuls les noms de lieux et les coordonnées sont envoyés pour récupérer des informations météo et géographiques publiques.

Analyse et Statistiques

L'application ne collecte PAS de statistiques d'utilisation, d'analyses ou de télémétrie. Toutes les informations restent sur votre appareil.

Cache et Données Temporaires

Pour améliorer les performances, l'application stocke temporairement des données météo et des informations de lieu localement. Ce cache :

  • Est stocké uniquement sur votre appareil
  • Est effacé automatiquement après un certain temps
  • Peut être effacé manuellement via les paramètres de l'application
  • Ne contient aucune information personnelle

Vos Droits

  • Vous pouvez supprimer toutes les informations stockées en désinstallant l'application
  • Vous pouvez effacer le cache et les fichiers de journal via les paramètres de l'application
  • Vous pouvez choisir de ne pas partager les fichiers de journal pour le support
  • Vous contrôlez quelles informations sont stockées localement
