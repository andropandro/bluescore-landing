Ihre Privatsphäre ist uns wichtig

Zuletzt aktualisiert: 26. Mai 2025

Überblick

BlueScore wurde mit Respekt für Ihre Privatsphäre entwickelt. Wir sammeln nur die Daten, die für die Funktionalität der App notwendig sind, und speichern alle Informationen lokal auf Ihrem Gerät.

Gesammelte Informationen

  • Ihre Ortssuchen
  • Wetterdaten für ausgewählte Orte
  • App-Einstellungen und Präferenzen
  • Technische Protokolldaten für Fehlersuche und Support

Nicht Gesammelte Informationen

  • Persönlich identifizierbare Informationen
  • Kontoinformationen oder Benutzerdaten
  • Bilder, Mediendateien oder persönliche Inhalte
  • Kontinuierliche Standortdaten im Hintergrund
  • Persönliche Nachrichten oder Kommunikation

Protokolldateien und Fehlersuche

Die App erstellt automatisch Protokolldateien auf Ihrem Gerät, um bei der Fehlersuche und dem Support zu helfen. Diese Protokolldateien enthalten:

  • Technische Informationen über die App-Funktionen
  • Fehlermeldungen und Leistungsdaten
  • Zeitstempel von Aktivitäten
  • Geräteinformationen (iOS-Version, App-Version)

Eine neue Protokolldatei wird bei jedem App-Start erstellt, und maximal fünf vorherige Protokolldateien werden automatisch gespeichert. Die Protokolldateien enthalten KEINE persönlichen Informationen wie Namen, besuchte Orte oder persönliche Einstellungen. Sensible Informationen wie API-Schlüssel werden automatisch herausgefiltert.

Teilen von Protokolldateien

Sie können freiwillig Protokolldateien mit unserem Support-Team über die App-Einstellungen teilen. Dies geschieht nur, wenn Sie aktiv eine E-Mail mit den angehängten Protokolldateien senden. Keine Protokolldateien werden automatisch oder ohne Ihre Zustimmung gesendet.

Datenverarbeitung

Alle Informationen werden lokal auf Ihrem Gerät gespeichert. Keine Daten werden an unsere Server gesendet oder mit Dritten geteilt, außer wenn Sie freiwillig Protokolldateien für den Support teilen.

Dienste von Drittanbietern

Wir nutzen die folgenden externen Dienste, um Wetter- und Ortsinformationen abzurufen:

  • MET Norway API (Wetterdaten)
  • GeoNames API (Ortsinformationen)
  • OpenStreetMap/Overpass API (geografische Daten)
  • WeatherAPI.com (zusätzliche Wetterdaten)
  • Apple Maps & CoreLocation (Ortsinformationen)
  • Andere öffentliche Datendienste

Diese Anfragen erfolgen ohne Weitergabe Ihrer persönlichen Informationen. Nur Ortsnamen und Koordinaten werden gesendet, um öffentliche Wetter- und geografische Informationen abzurufen.

Analysen und Statistiken

Die App sammelt KEINE Nutzungsstatistiken, Analysen oder Telemetrie. Alle Informationen bleiben auf Ihrem Gerät.

Cache und Temporäre Daten

Zur Verbesserung der Leistung speichert die App vorübergehend Wetterdaten und Ortsinformationen lokal. Dieser Cache:

  • Wird nur auf Ihrem Gerät gespeichert
  • Wird automatisch nach einer bestimmten Zeit gelöscht
  • Kann manuell über die App-Einstellungen gelöscht werden
  • Enthält keine persönlichen Informationen

Ihre Rechte

  • Sie können alle gespeicherten Informationen durch Deinstallation der App löschen
  • Sie können Cache und Protokolldateien über die App-Einstellungen löschen
  • Sie können wählen, keine Protokolldateien für den Support zu teilen
  • Sie kontrollieren, welche Informationen lokal gespeichert werden
